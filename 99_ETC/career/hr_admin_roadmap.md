## 2. 1~2년차 (실무 기초)

### 목표

- 업무 프로세스를 정확하게 수행합니다
- 노동법/사규 기본 지식을 습득합니다
- 빠른 대응과 꼼꼼한 기록을 체화합니다

### 인사 영역

| 업무      | 세부 내용                                 |
|-----------|-------------------------------------------|
| 근태 관리 | 출퇴근 기록, 연차 산정, 초과근무 집계     |
| 급여 보조 | 급여 데이터 입력, 4대보험 취득/상실 신고  |
| 채용 실무 | 공고 게시, 면접 일정 조율, 입사 서류 수합 |
| 퇴직 처리 | 퇴직금 산정 보조, 이직확인서 발급         |
| 교육 지원 | 법정의무교육 일정 관리, 참석자 취합       |

### 총무 영역

| 업무      | 세부 내용                                  |
|-----------|--------------------------------------------|
| 자산 관리 | 비품 발주, 재고 관리, 자산 대장 작성       |
| 시설 관리 | 사무실 유지보수 접수, 업체 연락            |
| 문서 관리 | 공문 수발신, 계약서 보관, 법인인감 관리    |
| 복리후생  | 경조사 처리, 건강검진 일정, 사내 행사 지원 |
| 차량/출장 | 법인차량 배차, 출장비 정산                 |

### 학습 필수

- 근로기준법 기초 (근로시간, 연차, 해고 요건)
- 4대보험 실무 (EDI 신고)
- Excel 중급 (VLOOKUP, 피벗테이블, 조건부 서식)
- ERP/HRIS 기본 조작 (사용하는 시스템)

## 3. 3~5년차 (독립 운영 + 개선)

### 목표

- 담당 업무를 독립적으로 완수합니다
- 기존 프로세스의 문제점을 발견하고 개선을 제안합니다
- 전문 영역(인사 OR 총무)을 선택합니다

### 인사 전문 트랙

| 영역 | 업무                                           |
|------|------------------------------------------------|
| 채용 | 채용 채널 관리, JD 작성, 면접 참여, 오퍼 레터  |
| 보상 | 급여 산정 전체, 연봉 테이블 관리, 성과급 설계  |
| 평가 | 평가 제도 운영, 등급 분포 관리, 피드백 취합    |
| 교육 | 사내교육 기획, 외부교육 예산 관리, 효과성 분석 |
| 노무 | 취업규칙 변경, 징계위원회 간사, 노동청 대응    |

### 총무 전문 트랙

| 영역 | 업무                                         |
|------|----------------------------------------------|
| 시설 | 사옥 이전, 인테리어 업체 선정, 유지보수 계약 |
| 구매 | 연간 구매 계획, 단가 협상, 공급업체 평가     |
| 안전 | 산업안전보건법 대응, 안전관리자 선임         |
| 보안 | 출입 시스템 관리, CCTV 운영, 정보보안 교육   |
| 대관 | 인허가 갱신, 관공서 신고, 사업자등록 변경    |

### 학습 필수

- 노동법 심화 (판례, 해고/징계 절차, 비정규직)
- 인사관리론 (직무분석, 역량모델)
- 데이터 분석 (이직률, 채용 전환율, 비용 분석)
- 자격증: 공인노무사(도전), PHR, 경영지도사

## 4. 5~7년차 (기획 + 제도 설계)

### 목표

- 인사/총무 제도를 기획하고 설계합니다
- 경영진에게 보고하고 의사결정을 지원합니다
- 부서 간 이해관계를 조율합니다

### 인사 기획

| 과제      | 산출물                               |
|-----------|--------------------------------------|
| 인력 계획 | 연간 인력 운영 계획 (TO 관리)        |
| 보상 설계 | 직급/직무별 급여 체계, 인센티브 설계 |
| 평가 제도 | OKR/KPI 체계 도입, 다면평가 설계     |
| 승진 제도 | 승진 기준, 직급 체계 재설계          |
| 조직 설계 | 조직도 변경, 부서 신설/통합 제안     |
| HRD 전략  | 핵심인재 프로그램, 리더십 교육       |

### 총무 기획

| 과제        | 산출물                             |
|-------------|------------------------------------|
| 예산 관리   | 부서 연간 예산 편성/집행/결산      |
| 사옥 전략   | 임대차 계약 전략, 원가 절감        |
| BCP         | 비상 대응 계획, 재난 대비 매뉴얼   |
| ESG         | 환경/안전 관리 체계, 탄소중립 계획 |
| 디지털 전환 | 전자결재, 스마트오피스 도입        |

### 학습 필수

- 전략적 인적자원관리 (SHRM)
- 재무/회계 기초 (예산 편성, 비용 분석)
- 프로젝트 관리 (일정/이해관계자 관리)
- 프레젠테이션/보고서 작성 역량
- 노사 관계 (단체교섭, 노동조합법)

## 5. 7~10년차 (전략 + 조직 설계)

### 목표

- 경영 전략과 연계된 인사/총무 전략을 수립합니다
- 팀을 이끌고 후배를 육성합니다
- 경영진의 전략적 파트너가 됩니다

### 핵심 과제

| 영역      | 내용                                  |
|-----------|---------------------------------------|
| HRBP      | 사업부별 인사 전략, 현업 리더 코칭    |
| 조직문화  | 핵심가치 내재화, 조직 진단, 변화 관리 |
| M&A 인사  | 인수합병 시 인력 통합, 조직 재편      |
| 글로벌 HR | 해외법인 인사 지원, 주재원 관리       |
| 총무 전략 | 지사 통폐합, 부동산 전략, 위기관리    |
| 리더십    | 팀 빌딩, 성과 관리, 동기 부여         |

### 학습 필수

- 경영학 (전략, 재무, 마케팅 기초)
- 코칭/퍼실리테이션
- 노사 전략 (교섭 전략, 쟁의 대응)
- MBA (선택) 또는 최고경영자과정

## 8. 실무 TIP

### 신입 시절 빠르게 인정받는 방법

- 요청 받기 전에 마감 일정표를 먼저 공유합니다
- 전화/메일 응대를 빠르고 정확하게 합니다 (경영지원은 속도가 신뢰)
- 엑셀 단축키(Ctrl+Shift+L, Alt+=, F4)를 외웁니다
- 사규/취업규칙을 전문 읽어두면 질문에 즉답이 가능합니다
- 경조사/생일 등 직원 이벤트를 챙기면 사내 평판이 빨리 올라갑니다

### 이직/성장 시그널

- 현재 회사에서 제도 변경 경험이 없으면 3~4년차에 이직을 고려합니다
- "운영만 하는 회사" vs "제도를 만드는 회사" — 후자에서 성장 속도가 다릅니다
- 100인 이상 기업은 노동법 이슈가 다양해서 학습 기회가 많습니다
- 스타트업 인사 경험은 폭넓지만, 대기업 인사 경험은 깊이가 있습니다

### 면접에서 자주 나오는 질문

- "연차 산정 시 입사일 기준과 회계연도 기준의 차이는?"
- "통상임금에 포함되는 항목과 제외되는 항목은?"
- "징계 절차에서 필수적으로 거쳐야 하는 단계는?"
- "52시간제 위반 시 사업주 책임과 벌칙은?"
- "퇴직금 중간정산 요건은?"

### 업무 자동화 아이디어

| 업무             | 자동화 방법                               |
|------------------|-------------------------------------------|
| 연차 현황 집계   | Excel 매크로 또는 Google Sheets 자동 알림 |
| 수습 평가 알림   | Outlook 반복 일정 + 템플릿 메일           |

| 법정교육 일정    | 캘린더 자동 등록 + 미참석자 알림          |
| 경조사 지급 현황 | ERP 자동 전표 연동                        |
