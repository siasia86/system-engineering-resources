## 2. 1~2년차 (전표 기초)

### 목표

- 복식부기 원리를 체화합니다
- 전표 입력/증빙 관리를 정확하게 수행합니다
- 일/월 마감 루틴을 익힙니다

### 업무

| 업무       | 세부 내용                                 |
|------------|-------------------------------------------|
| 전표 처리  | 매입/매출 전표, 일반전표, 대체전표 입력   |
| 증빙 관리  | 세금계산서, 계산서, 카드전표 수취/보관    |
| 자금 일보  | 입출금 현황, 통장 잔액 대사               |
| 채권/채무  | 거래처 원장 관리, 미수/미지급 현황        |
| 경비 정산  | 법인카드 정산, 출장비/교통비 처리         |
| 세금계산서 | 전자세금계산서 발행/수취, 부가세 대사     |
| 고정자산   | 자산 등록, 감가상각비 계산 보조           |
| 급여 회계  | 급여 전표, 4대보험 분개, 원천세 납부 보조 |

### 학습 필수

- 회계원리 (차변/대변, 분개 원리, 재무제표 구조)
- 부가가치세법 기초 (과세/면세, 매입세액공제)
- ERP 회계모듈 실습 (SAP, 더존, 영림원 등)
- Excel (SUM/SUMIF, 피벗, 대사표 작성)
- 법인세법 기초 (손금/익금 개념)

## 3. 3~5년차 (결산 + 세무 독립)

### 목표

- 월/분기/연간 결산을 독립적으로 마감합니다
- 부가세/법인세/원천세 신고를 직접 수행합니다
- 외부감사에 대응합니다

### 업무

| 영역      | 업무                                             |
|-----------|--------------------------------------------------|
| 결산      | 월 마감 체크리스트, 결산 조정분개, 재무제표 작성 |
| 부가세    | 예정/확정 신고, 영세율 증명, 대손세액공제        |
| 법인세    | 세무조정, 소득금액조정합계표, 법인세 신고        |
| 원천세    | 매월 원천세 신고, 연말정산, 지급명세서           |
| 감사 대응 | 감사인 자료 제출, 계정 분석, 주석 작성 보조      |
| 자금 관리 | 자금 계획, 단기 차입/상환, 어음 관리             |
| 지방세    | 취득세, 재산세, 지방소득세 신고                  |

### 학습 필수

- 재무회계 심화 (K-IFRS 주요 기준서)
- 법인세 실무 (세무조정, 이연법인세)
- 원가회계 기초 (제조원가, 공정별/주문별)
- 전산세무 1급 / 세무회계 2급
- 외부감사 대응 (감사증거, 확인서 절차)

## 4. 5~7년차 (관리회계 + 재무기획)

### 목표

- 경영진 의사결정을 지원하는 재무 정보를 생산합니다
- 예산 편성/관리, 원가 분석을 주도합니다
- 내부통제 체계를 설계합니다

### 업무

| 영역      | 업무                                          |
|-----------|-----------------------------------------------|
| 관리회계  | 부문별 손익, 제품별 원가, ABC 원가분석        |
| 예산 관리 | 연간 예산 편성, 실적 대비 분석, 롤링 포캐스트 |
| 재무 분석 | KPI 대시보드, 비율 분석, 동종업 벤치마크      |
| 내부통제  | SOX/내부회계관리제도 운영, 통제 테스트        |
| 자금 기획 | 중기 자금 수지 계획, 여유자금 운용 방안       |
| 연결 결산 | 연결 조정, 내부거래 제거, 연결 재무제표       |
| 세무 전략 | 이전가격, 조세감면, 세무 리스크 관리          |

### 학습 필수

- K-IFRS 심화 (금융상품, 리스, 수익인식)
- 관리회계 (CVP 분석, BSC, 투자 의사결정)
- 재무 모델링 (DCF, 시나리오 분석)
- 내부회계관리제도 (COSO 프레임워크)
- 자격증: AICPA, CPA(도전), 세무사(도전)

## 5. 7~10년차 (재무 전략)

### 목표

- 기업 전략과 연계된 재무 전략을 수립합니다
- 팀을 이끌고 투자/자금 의사결정에 참여합니다
- IR, M&A 재무 실사를 주도합니다

### 핵심 과제

| 영역        | 내용                                       |
|-------------|--------------------------------------------|
| 재무 전략   | 자본 구조 최적화, 배당 정책, 신용등급 관리 |
| IR          | 실적 발표, 투자자 미팅, 사업보고서 작성    |
| M&A 재무    | 밸류에이션, 재무 실사(FDD), 인수가 산정    |
| 자금 조달   | 회사채 발행, 유상증자, PF, 은행 차입 협상  |
| 글로벌 재무 | 해외법인 연결, 환 헤지, 이전가격 문서화    |
| 리더십      | 팀 관리, 후배 육성, 프로세스 표준화        |

### 학습 필수

- 기업재무 (Corporate Finance)
- 밸류에이션 방법론 (DCF, Comparable, Precedent)
- M&A 프로세스 (LOI → DD → SPA)
- 자본시장법 기초
- MBA (선택) / CFA (도전)

## 8. 실무 TIP

### 신입 시절 빠르게 인정받는 방법

- 전표 입력 속도를 높입니다 (ERP 단축키 필수 암기)
- 증빙 누락 건을 먼저 발견하고 보고합니다
- 마감일 2일 전에 미리 준비를 끝내놓습니다 (월초 3일 마감이면 전월 29일에 완료)
- 숫자 틀린 게 발견되면 즉시 보고합니다 (숨기면 신뢰 손실)
- 거래처 담당자와 원만한 관계를 유지합니다 (대사 시 협조 필수)

### 결산 마감 생존 팁

- 마감 체크리스트를 만들어두면 빠뜨리는 항목이 없습니다
- "기말 잔액 = 기초 잔액 + 증가 - 감소"를 항상 검증합니다
- 전기 대비 ±20% 이상 변동 항목은 미리 사유를 정리합니다 (감사 질문 대비)
- 세금계산서 수취 마감일(다음달 10일)을 놓치면 매입세액 공제 불가입니다
- 결산 조정 분개는 반드시 근거 서류를 첨부합니다

### 면접에서 자주 나오는 질문

- "매출채권 대손충당금 설정 방법은?"
- "감가상각 정액법과 정률법의 차이와 선택 기준은?"
- "부가세 예정신고와 확정신고의 차이는?"
- "법인세 세무조정에서 손금불산입 항목 예시는?"
- "IFRS와 K-GAAP의 주요 차이 3가지는?"
- "연결 결산 시 내부거래 제거 방법은?"

### 이직/성장 시그널

- 전표만 입력하는 단순 업무에서 벗어나지 못하면 2~3년차에 환경을 바꿉니다
- 결산을 처음부터 끝까지 해본 경험이 가장 큰 자산입니다
- 외부감사 대응 경험이 있으면 이직 시 우대 받습니다
- Big4 회계법인은 3~5년 경력 후 나와도 시장 가치가 높습니다

### 업무 자동화 아이디어

| 업무            | 자동화 방법                                  |
|-----------------|----------------------------------------------|
| 거래처 대사     | Excel Power Query로 ERP 데이터 자동 매칭     |
| 미수금 독촉     | ERP 연령 분석 + 자동 메일 발송               |
| 감가상각비 계산 | Excel 템플릿 (자산 등록 시 자동 산출)        |
| 부가세 대사     | 세금계산서 데이터 vs ERP 매출/매입 자동 비교 |
