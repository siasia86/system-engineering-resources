## 1. 저작권법 제9조

업무상 저작물의 저작자를 규정하는 조항.

```
법인·단체 그 밖의 사용자의 기획 하에 법인 등의 업무에 종사하는 자가
업무상 작성하는 저작물로서 법인 등의 명의로 공표되는 것의 저작자는
계약 또는 근무규칙 등에 다른 정함이 없는 때에는 그 법인 등이 된다.
```

회사 저작물로 인정되려면 아래 4가지 요건을 모두 충족해야 합니다.

1. 회사의 기획 하에 작성
2. 업무에 종사하는 자가 작성
3. 업무상 작성
4. 회사 명의로 공표

업무 외 시간에 작성했더라도 위 요건 중 일부를 충족하면 회사 저작물로 판단될 수 있습니다.

> 참고: [한국 저작권법 제9조](https://www.law.go.kr/법령/저작권법)

## 2. 근로계약서 귀속 조항

대부분의 IT 회사 근로계약서에는 아래와 유사한 조항이 포함됩니다.

```
"재직 중 업무와 관련하여 개발한 발명, 저작물, 기술 등의
지식재산권은 회사에 귀속된다."
```

**"업무와 관련하여"의 해석 범위가 핵심**입니다.
법원은 이 범위를 넓게 해석하는 경향이 있습니다.

## 3. 귀속 가능성 판단 기준

| 상황                                  | 회사 귀속 가능성 |
|---------------------------------------|------------------|
| 회사 업무와 무관한 개인 프로젝트      | 낮음             |
| 회사 기술 스택과 동일한 개인 프로젝트 | 중간             |
| 회사 업무 연장선의 개인 작업          | 높음             |
| 회사 장비/시스템 사용                 | 높음             |
| 근로계약서에 귀속 조항 존재           | 높음             |

## 4. 실무 대응

1. 근로계약서의 지식재산권 귀속 조항 범위 확인
2. 개인 프로젝트 공개 전 회사에 서면으로 확인
3. 회사 자원 미사용, 업무와 무관한 내용으로 명확히 분리
4. 퇴직 후 공개하더라도 재직 중 작성분은 동일하게 적용됩니다.
